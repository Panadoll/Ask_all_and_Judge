# Ask All & Judge (AI问个遍)

> 同时向所有 AI 发问，用评判 AI 对比回答 - 使用你自己的订阅，无需 API

## 🎯 为什么选择 AI问个遍？

不用再把同一个问题复制粘贴到多个 AI 标签页了。

**AI问个遍** 让你：
- 同时向多个 AI 发问，分屏查看所有回答
- 使用你现有的订阅（无需 API 密钥或额外费用）
- 用评判 AI 帮你分析对比所有回答

## ✨ 功能特性

| 功能 | 演示 |
|------|------|
| **🚀 并行查询**<br>同时向多个 AI 发送问题，并排查看所有回答。 | ![并行查询](../docs/demo-parallel-query.gif) |
| **⚖️ 评判 AI**<br>让评判 AI 分析对比所有回答，突出关键差异和见解。 | ![评判 AI](../docs/demo-judge.gif) |

| **📤 导出迁移**<br>导出对话为 Markdown 或在 AI 服务之间迁移回答。 | ![导出](../docs/demo-export.gif) |

## 🔧 安装

**从 Chrome 应用商店**（即将上线）

**手动安装**
1. 克隆此仓库
2. 在 Chrome 中打开 `chrome://extensions/`
3. 启用**开发者模式**
4. 点击**加载已解压的扩展程序**，选择 `ask-all-and-judge` 文件夹

## 🔒 隐私

- 无数据收集
- 使用你的浏览器会话
- 无需 API 密钥
- 完全在浏览器中运行
